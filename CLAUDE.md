# Global Coding Standards

Personal coding standards and conventions applied to all projects.

## Rules Overview

- **react.md** - Controller-view-hook pattern, folder structures, state management
- **typescript.md** - Type patterns and conventions
- **style.md** - Naming conventions and code style principles
- **documentation.md** - README maintenance guidelines

## Commands

- `/pr` - Generate PR description and create/update pull request
- `/check` - Run CI checks locally
- `/testing-plan` - Generate unit testing plan for a file
